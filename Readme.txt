# Weather App


Welcome to the Weather App! This web application allows you to easily search for weather information in different regions and provides details such as temperature, humidity, wind speed, sunrise, and sunset times. The app is built using HTML, JavaScript, and CSS.


## Features


- **Region Search**: Enter the name of a region to get real-time weather information.
- **Temperature**: View the current temperature in Celsius or Fahrenheit.
- **Humidity**: Check the humidity level of the selected region.
- **Wind Information**: Get details about the wind speed and direction.
- **Sunrise and Sunset Times**: Find out when the sun rises and sets in the selected region.


## Getting Started


To run the Weather App locally, follow these steps:


1. **Clone the Repository**: 
   ```bash
   git clone https://github.com/Mansh25/weather-app.git
   cd Weather_app




1. Open the App:
* Double-click on index.html to open the app in your web browser.


2. Usage:
* Enter the name of a region in the search bar.
* Press the "Search" button or hit "Enter".
* The weather information for the specified region will be displayed.


Technologies Used

* CSS: Styling to make the app visually appealing.
* JavaScript: Enables dynamic and interactive features.
